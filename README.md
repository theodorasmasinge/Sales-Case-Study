# Sales-Case-Study
# Overview of the Case Study
This case study analysed the Bright Car Sales dataset, which contained daily vehicle sales transactions, pricing information, and operational metrics.
The objective was to uncover insights that can help Bright Motors improve sales performance, understand trends, optimize pricing, and strengthen business strategy.
# How i did the Case study
Data Cleaning & Preparation:Cleaned inconsistent price values using numeric extraction.
Created additional calculated fields:Cost Price (80% of selling price), Profit Margin, Daily Gross Profit,Average Price per Unit and units sold
Standardized dates into correct Year, Month, and Day formats.
# Insights Found 
Sales Performance Peaked in 2014
Cost of Sales Is Increasing
Promo Days Contribute a Noticeable Portion of Sales
Quantity Sold Declined After 2014
